## 动词活用形态

- 未然形, 接续形一种
- 假定形, 接续形一种
- 命令形, 接续形一种
- 连用形
- 终止形
- 连体形

## 动词三分类

### 一段动词/上一段下一段动词

- 词尾有两个假名
- 词尾第一个假名是`い`段或`え`段
- 词尾第二个假名是`る`的动词

例如: いる、起きる、食べる

### 五段动词

- 词尾有一个假名
- 词尾的假名是`う`段非`る`
- 词尾的前一个假名非`い`段或`え`段

例如: 書く、出す、立つ

特例：きる、帰る、走る

### さ变/か变动词

- さ变: `する`结尾的动词
- か变只有一个动词`くる`

## 连用形变形

### 一段动词/上一段下一段动词

去除最后一个假名`る`

| 词例 | 连用形 |
| ---- | ----- |
| いる | い |
| 起きる | 起き |
| 食べる | 食べ |

### 五段动词

将词尾变为该行的`い`段假名

| 词例 | 连用形 |
| ---- | ----- |
| 書く | 書き |
| 出す | 出し |
| 立つ | 立ち |
| きる | きり |
| 帰る | 帰り |
| 走る | 走り |

### さ变/か变动词

- `さ`变将词尾`する`变为`し`

| 词例 | 连用形 |
| ---- | ----- |
| 勉強する | 勉強し |
| くる | き |

### 礼貌体

动词的连用形+`ます`组成礼貌体
